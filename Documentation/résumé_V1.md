# Resumé

## Comobil:

Comobil est une application qui répond au besoin de covoiturage orienté sur les trajets, l'objectif étant de permettre le partage de trajets entre plusieurs utilisateurs.

l'utilisateur s'inscrit et créer un trajet, qui est ensuite affiché à tous les utilisateurs cherchant le même trajet, pour s'organisé, les autres utilisateurs peuvent alors rejoindre le trajet pour le faire ensemble.

### Objectif Principal:

- les utilisateurs sont affiché par trajets commun, le matching se fait sur le point de départ et d'arrivée.

- les utilisateurs d'un trajet commun peuvent créer un trajet, qui est alors affiché à tous ceux qui font le même trajet

- les utilisateurs peuvent rejoindre un trajet commun existant

## MVP :
pour le mvp, le logiciel se contentera d'abord de matcher les points de départ et d'arriver et de founir la liste des trajets

### cible du mvp:
- particuliers disposant d'un véhicule ou non.

#### fonctionnalité principale:

-si il y a 10 utilisateur qui partent à toulouse, 5 qui partent à Montauban, 5 qui partent à Albi, alors les utilisateurs sont groupé par ville de destination sans se préoccupé de savoir quelles autre ville sont traversée, ni à chercher des trajets optimisé pour chaque utilisateur.

## évolution possible:

les utilisateurs mettent à disposition leurs véhicule, précise le type de trajet (régulier ou occasionel), ils précisent l'horraire, les lieux de rendez vous pour le départ et le retour, être notifié quand un utilisateur rejoint leurs trajets commun.

### cible de l'évolution:
particulier
entreprises
maire/commune
